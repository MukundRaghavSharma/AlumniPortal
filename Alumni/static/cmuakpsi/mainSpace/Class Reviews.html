Unofficial Reviews
The class reviews are a way for us to create a consolidated list of all the classes brothers have taken at CMU, and how we felt about them in terms of difficulty and usefulness. This can be useful in creating a balanced schedule, finding breadth requirements that won't be too difficult, and knowing what to expect as we begin each semester. We encourage everyone to contribute any class they've taken to this list, even if your class is already on the list. The more data the better.
